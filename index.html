Re-enable motion access

You denied motion access, and iOS won't ask again until this site's saved data is cleared. Pick whichever is easiest, then reload and tap Activate again.

Easiest: open this page in another browser, or in a Private tab.

Other ways

Safari only — clear this site's data:

Quick — clears the last hour of history:
Settings → search Clear History and Website Data → Clear History → choose Last hour.

Precise — removes only this site:
Settings → Apps → Safari → Advanced → Website Data → search for this website → tap the Search button or hide the keyboard to reveal Edit → Edit → tap the − next to it → Delete.

Other browsers — look for an option to clear browsing history, or reinstall the browser app.
